ViewLingo에 대해 알아야 할 모든 것

🚀 시작하기

뷰파인더를 어떻게 생성하나요? ▼
뷰파인더를 생성하는 가장 빠른 방법:
  • 즉시 생성: fn+Control+Option을 눌러 마우스 위치에 즉시 뷰파인더를 생성합니다
  • 대안: 메뉴 바 아이콘이나 앱 메뉴를 사용해서도 뷰파인더를 생성할 수 있습니다
뷰파인더는 이동하고 크기를 조절할 수 있는 반투명 창으로 나타납니다.
뷰파인더를 어떻게 닫나요? ▼
여러 가지 방법으로 뷰파인더를 닫을 수 있습니다:
  • ESC 키 누르기
  • 뷰파인더 아무 곳이나 트리플 클릭
  • 모서리의 닫기 버튼(×) 클릭
ViewLingo는 어떤 권한이 필요한가요? ▼
ViewLingo는 뷰파인더 내의 콘텐츠를 캡처하기 위해 화면 녹화 권한이 필요합니다. 이는 화면 콘텐츠를 캡처하는 모든 앱에 대한 macOS 요구사항입니다.
프라이버시 참고: ViewLingo는 뷰파인더 내의 특정 영역만 캡처하며 완전히 기기에서 처리합니다. 콘텐츠는 Mac을 떠나지 않습니다.

🌍 언어 및 번역

어떤 언어를 지원하나요? ▼
ViewLingo는 다음 10개 언어 간 번역을 지원합니다:
  • 영어 (English)
  • 한국어
  • 일본어 (日本語)
  • 중국어 간체 (简体中文)
  • 중국어 번체 (繁體中文)
  • 스페인어 (Español)
  • 프랑스어 (Français)
  • 독일어 (Deutsch)
  • 러시아어 (Русский)
  • 이탈리아어 (Italiano)
  • 포르투갈어 (Português)
번역 정확도는 어떤가요? ▼
ViewLingo는 Apple의 내장 신경망 번역 엔진을 사용하여 다른 주요 번역 서비스와 비슷한 수준의 고품질 번역을 제공합니다. 정확도는 다음에 따라 달라집니다:
  • 텍스트 선명도와 글꼴 품질
  • 언어 쌍 (일부 조합은 더 잘 작동합니다)
  • 문맥과 텍스트 유형 (간단한 문장 vs. 기술적 내용)
최상의 결과를 위해 텍스트가 화면에 선명하게 보이도록 하세요.
오프라인에서도 번역할 수 있나요? ▼
네! ViewLingo는 완전히 오프라인에서 작동합니다. 모든 번역은 Apple의 내장 번역 엔진을 사용하여 기기에서 처리됩니다. 앱이 설치되면 인터넷 연결이 필요하지 않습니다.

⚡ 기능 및 성능

실시간 번역 모드란 무엇인가요? ▼
라이브 모드는 화면의 콘텐츠가 변경될 때 자동으로 재번역합니다. 다음과 같은 경우에 완벽합니다:
  • 비디오 자막
  • 라이브 스트림
  • 동적 웹 콘텐츠
  • 스크롤되는 문서
⚠️ 성능 경고: 라이브 모드는 메모리 사용량을 크게 증가시키며 시스템 성능에 영향을 줄 수 있습니다. 기본적으로 비활성화되어 있으며 장시간 사용 시 주의가 필요합니다.
ViewLingo가 Mac을 느리게 만드나요? ▼
일반적인 사용에서 ViewLingo는 시스템 성능에 최소한의 영향을 미칩니다:
  • 일반 모드: 매우 낮은 리소스 사용량, 번역 요청 시에만 처리
  • 라이브 모드: 지속적인 처리로 인한 높은 리소스 사용량
이 앱은 macOS 15.0+에 최적화되어 있으며 Apple Silicon과 Intel Mac 모두에서 효율적으로 작동합니다.
모든 앱에서 텍스트를 번역할 수 있나요? ▼
네! ViewLingo는 다음을 포함한 모든 애플리케이션이나 웹사이트의 텍스트를 번역할 수 있습니다:
  • 웹 브라우저 (Safari, Chrome, Firefox)
  • PDF 리더
  • 비디오 플레이어
  • 게임
  • 문서 및 프레젠테이션
  • 소셜 미디어 앱
뷰파인더는 소스 애플리케이션에 관계없이 화면에 표시되는 모든 것을 캡처합니다.

💰 가격 및 구매

ViewLingo가 왜 $4.99인가요? ▼
ViewLingo는 다음을 포함하는 일회성 구매입니다:
  • 제한 없는 모든 기능
  • 10개 언어 팩 모두
  • 평생 업데이트
  • 구독이나 반복 요금 없음
  • 온디바이스 처리로 프리미엄 프라이버시
대부분의 번역 구독 서비스 한 달치보다 저렴하지만 영원히 소유할 수 있습니다.
가족과 ViewLingo를 공유할 수 있나요? ▼
네! ViewLingo는 Apple의 가족 공유를 지원하므로 추가 비용 없이 최대 6명의 가족 구성원과 구매를 공유할 수 있습니다. 또한 모든 개인 Mac에 설치할 수 있습니다.
ViewLingo가 만족스럽지 않으면 어떻게 하나요? ▼
ViewLingo는 Apple의 표준 환불 정책을 따릅니다. 앱이 요구사항을 충족하지 못하면 구매 후 14일 이내에 Apple을 통해 환불을 요청할 수 있습니다. 환불 처리를 위해 Apple 지원에 문의하세요.

🔒 프라이버시 및 보안

ViewLingo에서 내 데이터는 안전한가요? ▼
물론입니다! ViewLingo는 프라이버시를 핵심 원칙으로 설계되었습니다:
  • 모든 번역은 기기에서 처리됩니다
  • 외부 서버로 데이터가 전송되지 않습니다
  • 콘텐츠를 수집, 저장, 전송하지 않습니다
  • 분석이나 추적이 없습니다
  • 완전히 오프라인으로 작동합니다
번역은 생각만큼 프라이빗합니다 - Mac을 떠나지 않습니다.
화면 녹화 권한은 어떤가요? ▼
화면 녹화 권한은 ViewLingo가 뷰파인더 내의 콘텐츠를 캡처할 수 있게 합니다. 이는:
  • 모든 화면 캡처 기능에 대한 macOS 요구사항입니다
  • 선택한 특정 영역으로 제한됩니다
  • 즉시 처리되고 저장되지 않습니다
  • 원격으로 액세스하거나 전송할 수 없습니다
ViewLingo는 이 권한을 책임감 있게 번역 목적으로만 사용합니다.

🛠️ 문제 해결

ViewLingo가 텍스트를 제대로 인식하지 못합니다. 어떻게 해야 하나요? ▼
다음 해결책을 시도해보세요:
  • 텍스트 선명도 확인: 텍스트가 선명하게 보이고 흐릿하지 않은지 확인
  • 뷰파인더 크기 조정: 전체 텍스트가 뷰파인더 내에 있는지 확인
  • 조명 개선: OCR은 좋은 대비에서 더 잘 작동합니다
  • 언어 설정 확인: 소스 언어가 지원되는지 확인
  • 다른 글꼴 시도: 일부 장식 글꼴은 인식하기 어려울 수 있습니다
앱이 충돌하거나 멈춥니다. 어떻게 고치나요? ▼
  1. ViewLingo 재시작: 앱을 종료하고 다시 실행
  2. macOS 버전 확인: macOS 15.0 이상을 실행 중인지 확인
  3. 라이브 모드 비활성화: 활성화된 경우 설정에서 라이브 모드를 끄기
  4. Mac 재시작: 때때로 시스템 재시작이 도움이 됩니다
  5. 재설치: App Store에서 삭제하고 재설치
문제가 지속되면 GitHub Issues 페이지에 문제를 보고해주세요.

문제를 재현하기 위한 macOS 버전과 자세한 단계를 포함해주세요.
