Live Mode automatically retranslates content as it changes on screen, perfect for:
  • Video subtitles
  • Live streams
  • Dynamic web content
  • Scrolling documents

ViewLingo isn't recognizing text properly. What can I do? ‚ñº
Try these solutions:
  • Check text clarity: Ensure the text is clearly visible and not blurry
  • Adjust viewfinder size: Make sure the entire text is within the viewfinder
  • Improve lighting: OCR works better with good contrast
  • Check language settings: Ensure the source language is supported
  • Try different fonts: Some decorative fonts may be harder to recognize
The app crashes or freezes. How do I fix this? ‚ñº
  1. Restart ViewLingo: Quit and relaunch the app
  2. Check macOS version: Ensure you're running macOS 15.0 or later
  3. Disable Live Mode: If enabled, turn off Live Mode in settings
  4. Restart your Mac: Sometimes a system restart helps
  5. Reinstall: Delete and reinstall from the App Store
If problems persist, please report the issue on our GitHub Issues page.
