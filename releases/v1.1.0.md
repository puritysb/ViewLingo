# ViewLingo 1.1.0 - Performance Update

Major performance improvements and bug fixes.

## Performance Improvements

- **5x faster Live Mode** response time for video and streaming content
- **60% reduced memory usage** during extended translation sessions
- Better handling of streaming and dynamic content

## Bug Fixes

- Fixed Japanese text detection in Live mode
- Fixed viewfinder getting stuck after stopping Live mode
- Fixed infinite loading on YouTube and video streaming sites
- Improved keyboard shortcut reliability
- Fixed button visibility in dark mode
- Fixed text truncation in Korean and Japanese onboarding

## System Requirements

- macOS 15.0 (Sequoia) or later
- Screen Recording permission required
